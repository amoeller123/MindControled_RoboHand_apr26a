Python script based off of https://pypi.python.org/pypi/NeuroPy/0.1 and designed to work with a Raspberry Pi.
A Rocketfish Micro Bluetooth USB Adapter was used to interface with a Neurosky Mindwave Mobile EEG headset. 
Rocketfish uses a Broadcomm chipset with BlueZ Linux Bluetooth protocol stack installed on the Raspberry Pi.
